About this program

This program is an excerpt from a larger program designed to manage the production of cutting forme manufacturing.

For demonstration purposes, the page has been heavily modified as the original program requires a connection to a database, which this demo cannot have. Due to this drawback, many values originally retrieved from the database have been hard-coded and slightly altered for use in this program.

The aspects of the program that have been altered from the original are the retrieval of a customer's delivery offset values. These are values that define how many days the order needs to be completed before it is despatched to the customer. These are typically retrieved from the database upon selecting a customer, however they have been hard-coded and altered for the sake of this demo.

This page is made up of code adapted from my in-house "War Room" Job Management system.
